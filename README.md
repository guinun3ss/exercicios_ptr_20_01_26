# exercicios_ptr_20_01_26
Implementações referentes aos exercícios de Estruturas de Dados I sobre ponteiros do dia 20/01/2026.

1. Criar um programa em C que receba varáveis de diversos tipos e ponteiros apontando para elas e imprima seus dados e seus endereços de memória utilizado os ponteiros.

2. Criar um programa em C que receba um vetor de valores reais e imprima esse vetor com o uso de ponteiros.

3. Criar um programa em C que altere o código da questão anterior fazendo com que o usuário informe tanto o tamanho do vetor quanto os valores armazenados nele e continue imprimindo com ponteiros.

4. Criar um programa em C que faça o mesmo da questão anterior utilizado função com passagem de parâmetro utilizando o vetor, tanto para imprimir quanto para alterar.

5. Criar um programa em C que receba uma Struct com dados de um cliente, altere e imprima ela utilizando ponteiros.

6. Criar um programa em C que receba um vetor de valores inteiros, identifique o maior e o menor valor e mostre seus  seus valores com ponteiros.

7. Crie um programa em C que tenha uma função que use ponteiros para trocar os valores de duas valores.

8. Crie um programa em C que contenha um vetor com 10 posições de qualquer tipo (integer, float ou char) e imprima o endereço de memória e seu conteúdo.

9. Crie um programa em C que copie um vetor para outro vetor. (dica - utilize função e ponteiro)

10. Com a mesma lógica da questão anterior, faça isso com uma string.
